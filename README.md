# 재난지역 탐사 로봇 GAEMI

---

## 프로젝트 개요

* **SSAFY 12기 자율PJT**
* **결선 발표 전체 1위**

## 전체 흐름

1. **로봇** (Hexapod 6족 보행 로봇)
2. **Controller** (컨트롤러)
3. **Dashboard** (대시보드)

## 개발 부분

* **로봇 컨트롤 PC (젯슨 나노) 펌웨어 설정**
* **로봇 ⇔ 컨트롤러 통신 및 제어**

  * 틸팅(Tilting)
  * 업다운(Up/Down)
  * 인사(Greeting)
  * 밸런스(Balance)
* **컨트롤러 코드 통합**
* **컨트롤러 ⇔ 대시보드 통신**

  * MQTT
  * RTMP
